# 燃速誌

燃速誌是一個以個人使用為主的跑步紀錄網頁，結合配速計算、跑步資料管理、統計分析、月曆檢視、成就系統與深淺色模式。它的目標不是做成複雜的社群平台，而是讓使用者能快速記下每次跑步，並清楚看到自己的累積進步。

## 功能特色

### 配速計算

- 支援常用距離與自訂距離。
- 可輸入時間後自動計算配速。
- 提供配速與時速轉換。
- 可將計算結果帶入跑步紀錄。

### 跑步紀錄

- 可記錄日期、名稱、室內／室外、距離、時間、平均配速與每公里配速。
- 可補充平均心率、最大心率、卡路里、步頻與備註。
- 支援心率區間 Z1 到 Z5 的比例紀錄。
- 可記錄天氣資訊，包含氣溫、濕度、風速與風向。
- 每筆紀錄可展開或收合，方便瀏覽大量資料。

### 統計與最佳紀錄

- 顯示本週、本月與今年跑量。
- 顯示最快配速、最長距離與最長時間。
- 提供本月跑量趨勢與連續跑步狀態。
- 支援搜尋、篩選、排序與顯示筆數選擇。

### 月曆檢視

- 以月曆方式呈現每天的跑步紀錄。
- 有跑步的日期會依照距離深淺顯示，方便快速看出訓練分布。
- 可切換不同月份查看過去紀錄。

### 成就系統

- 依照距離、累積等級、次數、連續跑步、配速、挑戰、耐力與最佳紀錄解鎖成就。
- 每個成就會顯示達成日期。
- 累積距離採用等級制，讓長期累積更有目標感。
- 另有隱藏成就，增加一點探索感。

### 個人檔案

- 可填寫名字、歲數與目標。
- 顯示目前累積等級、累積距離與下一級進度。
- 背景色會依照累積等級變化，讓成長狀態更直覺。

### 主題與互動

- 支援淺色與深色模式。
- 深色模式以黑色為主要背景，適合夜間使用。
- 展開、切換、懸停與數字變化都有細緻動畫，讓操作更順暢。

## 資料保存

燃速誌目前是純前端網頁，資料會保存在瀏覽器本機中。只要使用同一個瀏覽器與同一個網址，重新整理或下次開啟時資料會保留。

如果換裝置、換瀏覽器、清除瀏覽器資料，原本資料可能不會自動出現。因此建議定期使用內建的備份功能下載 JSON 檔，需要時再用匯入功能還原。

## 匯出與備份

- 備份：下載完整資料，適合日後還原。
- 匯入：將備份檔重新匯入網站。
- CSV：匯出跑步紀錄，方便用試算表整理或保存。

## 適合誰使用

燃速誌適合想要手動整理跑步資料的人，尤其是平常會從不同 App 或截圖中查資料，但希望最後能集中到一個乾淨、快速、可長期保存的地方。

它適合記錄：

- 日常慢跑
- 配速訓練
- 月跑量目標
- 個人最佳紀錄
- 長期累積里程與成就

## 專案定位

這是一個個人跑步 Dashboard。它重視的是紀錄速度、閱讀清楚、資料可控與持續使用的動力。未來可以繼續擴充更多分析功能，但核心仍會維持簡單、直接、以個人訓練紀錄為主。
